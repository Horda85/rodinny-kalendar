🏡 Rodinný Kalendář

Propojeno s Google Calendar. Každý člen rodiny může přidávat a editovat události ze svého zařízení.

1

Vytvoř API přihlášení

Přejdi na Google Cloud Console → APIs & Services → Credentials → Create Credentials → API Key. Povolь "Google Calendar API".

2

Vytvoř OAuth 2.0 Client ID

V Credentials → Create Credentials → OAuth Client ID (typ: Web application). Do "Authorized JavaScript origins" přidej URL kde soubor hostíš (nebo http://localhost pro lokální test).

3

Zadej přihlašovací údaje níže

API Key a Client ID zkopíruj a vlož sem. Uloží se lokálně v prohlížeči.
